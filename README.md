This project contains:

1. scraper.py
   - Scrapes first 100 games from given website
   - Saves data into CSV file

2. rag.py
   - Creates simple vector database using FAISS
   - Allows querying similar game descriptions

How to run:

1. python scraper.py
2. python rag.py
